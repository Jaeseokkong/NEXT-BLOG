## 프로젝트 개요

**WebSocket 기반 실시간 VOC 데이터 시각화 어플리케이션**을 개발했습니다.  
기기에서 발생하는 로그 데이터를 실시간으로 수집하고, 관리자 페이지에서 직관적인 그래프와 차트로 보여줍니다.

## 주요 기능

- 기기에서 발생한 로그 데이터를 실시간으로 수집 및 처리
- 관리자 페이지에서 실시간 그래프 및 차트로 데이터 시각화
- 사용자 피드백을 반영한 UI/UX 개선으로 편의성 향상
- VOC 데이터의 즉각적 활용으로 빠른 대응 지원

## 주요 성과

- 실시간 데이터 처리로 VOC 문제 신속 인지 및 해결 가능
- 관리자 UI 개선을 통한 업무 효율성 증대
