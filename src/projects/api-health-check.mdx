회사의 내부 API 상태를 실시간으로 모니터링할 수 있는 **대시보드형 헬스 체크 시스템**을 개발했습니다.

- `Next.js` 기반 CSR로 구성해 대시보드 퍼포먼스 최적화
- API 응답 속도, 상태 코드, 장애 유무 등을 실시간 그래프로 시각화
- 장애 발생 시 빠른 인지 가능 → 운영팀과 개발팀 대응 시간 단축
- 시스템 신뢰도 및 운영 효율성 향상
