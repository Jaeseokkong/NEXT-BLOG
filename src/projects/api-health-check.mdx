## 프로젝트 개요
회사의 내부 API 상태를 실시간으로 모니터링할 수 있는 **대시보드형 헬스 체크 시스템**을 구축했습니다.  
이를 통해 API의 안정성과 가용성을 체계적으로 관리하며, 장애 대응 속도를 획기적으로 개선하는 것을 목표로 했습니다.

## 주요 기능
- `Next.js` 기반 클라이언트 사이드 렌더링(CSR)으로 대시보드 퍼포먼스 최적화  
- API 응답 시간, 상태 코드, 장애 발생 여부 등 핵심 지표를 실시간 그래프로 시각화  
- 장애 발생 시 빠른 인지 및 알림 기능을 통해 운영팀과 개발팀의 대응 시간 단축  
- 시스템 전반의 신뢰도 향상과 운영 효율성 증대에 기여  
- Linux 서버 환경에서 안정적으로 서비스 운영 및 유지보수 수행
