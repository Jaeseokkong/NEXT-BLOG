## 프로젝트 개요
교사용 웹 기반 학습자 관리 플랫폼의 신규 기능 개발과 기존 시스템 유지보수를 담당했습니다. 학습자 관리부터 성적 평가, 피드백 제공까지 통합적으로 지원하는 플랫폼으로, 교사의 업무를 효율적으로 돕기 위한 다양한 기능들을 구현했습니다.

## 주요 기능
- 학습자 정보 관리, 성적 평가, 피드백 제공 기능을 구현하여 교사의 업무를 지원
- 대시보드 시각화 기능 추가로 학습자 성취도와 진척도를 한눈에 파악 가능
- 학습 리포트 PDF 다운로드 기능 개발로 교사 및 학부모 대상 보고서 제공 편의성 강화
- 사용자 피드백을 반영하여 UI 오류 수정 및 인터페이스 개선, 사용자 경험 향상

## 주요 성과
- 시각화 및 리포트 기능 도입으로 교사 업무 효율이 약 15% 향상
- 플랫폼 안정성 강화 및 사용자 만족도 증가
- Azure 기반 인프라 환경에서 안정적인 서비스 운영과 유지보수 수행
